# electro-network-api
Веб-приложение на Django + DRF для моделирования и управления иерархической сетью по продаже электроники (заводы → розничные сети → ИП) с админ-панелью, API, фильтрацией по стране и правами доступа для активных сотрудников. Поддерживает учёт поставщиков, продуктов, задолженности и удобные действия в админке
